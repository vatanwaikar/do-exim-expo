## 🚀 Deployment Instructions

### Local Development
```bash
cd /vercel/share/v0-project
pnpm install
pnpm dev
# Opens at http://localhost:3000
```

### Production Build
```bash
pnpm build
pnpm start
```

### Deploy to Vercel
1. Push code to GitHub
2. Connect repository to Vercel
3. Click Deploy
4. Auto-deploys on every push
